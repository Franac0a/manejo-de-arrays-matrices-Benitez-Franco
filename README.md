## Cómo ejecutar

🔹 JavaScript (Node.js)

Este programa utiliza la librería prompt-sync.

Instalar dependencias:

npm install

Si no funciona, usar:

npm install prompt-sync

Ejecutar:

node index.js
🔹 Python

No requiere instalación de librerías.

Ejecutar:

python main.py

- Finalizar carga de datos

Para terminar el ingreso, escribir:

fin
Salida del programa

El sistema mostrará:

Lista de personas ingresadas

Lista ordenada por nota (de mayor a menor)

Promedio general de notas
